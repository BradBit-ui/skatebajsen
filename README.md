# 💩 Skate-Bajsen

Ett fartfyllt retro-pixelspel där du styr en skejtande bajskorv genom en färgstark stad! Samla magiska föremål, undvik ingenting (för allt är bra!), och se hur världen ändrar färg ju bättre du blir.

## 🚀 Spela nu
Spelet finns tillgängligt via GitHub Pages:
[https://BradBit-ui.github.io/skatebajsen/](https://BradBit-ui.github.io/skatebajsen/)

## 🎮 Hur man spelar

### På dator (Desktop)
- **Hoppa:** Tryck på **Space** (mellanslag) eller klicka med musen var som helst på skärmen.
- **Mål:** Samla så många enhörningar, stjärnor och hjärtan du kan!

### På mobil & surfplatta
- **Hoppa:** Tryck var som helst på skärmen (Touch).
- Spelet är optimerat för att fungera smidigt på både iOS och Android.

## ✨ Funktioner
- **Dynamisk svårighetsgrad:** Spelet blir snabbare och svårare ju fler poäng du samlar.
- **Level-system:** Himlen byter färg för varje 5:e föremål du plockar upp.
- **Boss-föremål:** Var 7:e föremål är extra stort och ger ett speciellt ljud ifrån sig!
- **Ljudeffekter:** Innehåller klassiska "fart"-ljud, enhörningsgnägg och peppande röster.
- **Pixel Art:** Handgjord grafik med transparenta lager för en äkta retrokänsla.

## 🛠 Teknik
Spelet är byggt med:
- **HTML5 Canvas** för grafikrendering.
- **Vanilla JavaScript** för spelmotor och logik.
- **CSS3** för responsiv design och den snygga rosa pixel-ramen.
